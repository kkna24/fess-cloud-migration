## Project Entry

**FESS Cloud Migration — Data Migration Specialist** | Seneca Polytechnic, 2024

- Migrated a MongoDB log database (NoSQL) to **Amazon DynamoDB** by building a Python script using `boto3` that batch-imports JSON exports, with error handling and retry logic for failed records
- Migrated a SQL Server (MSSQL) application database to **Amazon RDS** by writing the schema DDL and an automated shell script using `sqlcmd` to provision and verify tables on the remote instance
- Designed the data migration layer of a three-tier serverless AWS architecture (Lambda, API Gateway, DynamoDB, RDS, Cognito, SNS) replacing a legacy on-premises system
- Evaluated AWS Database Migration Service (DMS) and implemented a manual export/import alternative when IAM policy restrictions blocked direct access — delivered the same outcome without elevated permissions
- Documented the full migration process including architecture diagrams, step-by-step runbooks, and a cost/ROI analysis demonstrating cloud savings over on-premises hosting
